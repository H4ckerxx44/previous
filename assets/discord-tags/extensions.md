**Built-in:**
- ext.commands: <https://github.com/nextcord/nextcord/tree/master/nextcord/ext/commands>
- ext.tasks: <https://github.com/nextcord/nextcord/tree/master/nextcord/ext/tasks>

**Install separately:**
- ext.menus: <https://github.com/nextcord/nextcord-ext-menus>
-- install: `pip install -U nextcord-ext-menus`
- ext.ipc: <https://github.com/nextcord/nextcord-ext-ipc>
-- Install: `pip install -U nextcord-ext-ipc`

**Importing:**
- `from nextcord.ext import ...`
E. g, `from nextcord.ext import menus`
